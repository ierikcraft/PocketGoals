## Pocket Goals

 - **Pocket Goals** is an app made for you to do your money goals, like paypal but not real money and for goals.
 - You can make many as fake account you want even pockets.
 - This is inspired with the pockets function of the <18 revolut app

> [!TIP]
> This is open source, so you can use it for anything you want and modify as long that you want, its ready to be published, no configurations.
